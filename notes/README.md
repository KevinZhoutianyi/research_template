# Notes

## Subdirectories

- **writeups/** — Project formulations, reports, and planning documents.
- **read/** — Summaries of papers, written by Claude for reference.
- **weekly_updates/** — Beamer slides and feedback from each weekly meeting. Each subfolder is `YYYY-MM-DD/` and contains `slides.tex`, all images, and `feedback.md`.
- **discussions/** — Raw meeting notes. These are noisy and should not be read by Claude unless explicitly requested.
- **other/** — Anything that doesn't fit the above categories.

## Purpose

This folder is not just for human use — it serves as shared context for Claude. When working on the project, Claude can reference these notes to stay aligned on project goals, prior decisions, and relevant literature.
